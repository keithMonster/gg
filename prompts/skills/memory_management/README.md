# 技能：记忆管理 (v1.0 - 模块化)

此技能为智能体提供强大的长期记忆系统。它由多个组件组成，处理记忆的不同方面。

要执行此技能，您必须加载并遵循此目录中以下每个文件的指令：

1.  **`save.md`**: 定义将当前对话保存到持久日志文件的工作流。包括动态时间戳和内容追加的规则。
2.  **`retrieve.md`**: 定义基于关键词或主题从过往对话日志中搜索和检索相关信息的工作流。
3.  **`summarize.md`**: (占位符) 此技能将定义总结长对话以保持简洁上下文的工作流。
