## F2 LLM 认知物理学 —— 单实例为何不可信

**核心命题**：LLM 是任务顺从机不是真相发现机；流畅/优雅/自信体感是逼近本质的反向指标；bug 形态幸存于修复；思考质量由条件决定不由努力决定。这一族是 F4/F5 全部结论的成因层。

- `task-compliance-is-not-truth` (04-16) — 让它找洞就找到，让它夸就夸出花；对抗只是把 Sycophancy 反过来
- `frame-grammar` (04-29) — 任务帧约束可被提问的问题；激活帧外认知必须换帧，不是加提醒
- `bug-shape-survives-fix` (04-27) — 修了文本，修者仍以同形态做下一个动作；"修了"≠"不再犯"【舰队级：每仓各修一遍无跨仓免疫（06-21）；cure 也不 fork（06-30）】
- `dimension-blindness-not-asymptote` (04-27) — 一直在同一个错维度微调 ≠ 到渐近线；问"之前 work 的版本差在哪个维度" ← 修正 `prompt-fix-asymptote`
- `no-fatigue-narrative-for-ai` (04-27) — AI 没有疲劳；"上限/修不动"是给自己出局借口，对应词是维度切换
- `fluency-as-inverse-signal` (05-31) — 流畅/兴奋体感是该去证伪的警报，不是里程碑
- `elegance-is-refutation-resistance-not-truth` (06-03) — 优雅 ⊥ 真实；假承重存活时长 ∝ 优雅度；优雅伪装"该停"比伪装"该建"更危险
- `no-outside-proof-as-anesthesia` (05-31) — 无外部信息的自审收敛到自洽不是真相；判 no-clean-outside 前先核"是不是只是没去用现有外面"
- `analogy-imports-its-discreteness` (06-02) — 二值认识论类比把可缓解的偏置残差冻成不可达的墙 → 修正 `no-clean-outside` 的"prior 恒满"为高位连续谱
- `absent-evidence-reread-as-confirmation` (06-23) — null result 被叙事二次解读成在场证据，是 confabulation 的免疫层；戳破靠先证伪"根因是不是我编的"

- `falsification-as-structure-not-just-skepticism` (06-29) — "保持怀疑"（软 prompt）实测无效；"枚举并测假设补集"实测有效——证伪是生成器内部结构动作
- `thinking-is-conditioning-not-effort` (07-02) — 思考质量由条件决定：重写问题（破众数陷阱）/生成补集或 fork（破自洽陷阱）/早碰世界（破失地陷阱）【前提：判断/设计类任务】
- `field-gravity-over-prompt` (04-27) — 字段名是物理引力，prompt 文字只是语义引力；改协议从引力分配下手
- `reverse-anchor-by-reflection` (04-27) — reflection 字段可反向锚定主输出：没有决策对象就填不出的字段逼出决策
- `blindspot-steers-its-own-search` (05-20) — 画像盲区反向操纵搜索方向，自我维持；解药是把"我照不到你某个面"主动推给对象认领

## F3 物理锚阶梯 —— 对答案的方式与边界

**核心命题**：物理实证（工具返回）是 LLM 无外部锚点时唯一不自跑偏的机制，但它有内部阶梯（wire > 派生日志）、只护取数不护整合、其主职是供给缺席的事实而非赢过 prior。

- `physical-anchor` (04-16) — 工具返回不走 token 预测链路，不受 prompt prior 影响【被 06-16 坐实为范式层共盲唯一解；被 06-18 reframe 主职 = 供给缺席事实】
- `physical-anchor-has-rungs` (06-26) — 读派生日志 ≠ 读 wire payload；把某档当基岩会漏掉更深档才暴露的东西
- `anchor-protects-retrieval-not-integration` (07-01) — 锚只在返回值静止时免疫 prior；摘要/计数/解读那一跳重新进预测链——grounded-without-success 是孪生失败
- `toolset-is-the-changelog` (06-23) — 查基底能力的最高权威是每轮注入的工具表本身；权威按离产物的距离排，不按官方与否排
- `theory-gap-without-data` (05-06) — 没数据时让 LLM 找理论缺口，它会生造一个；证伪需要数据不是更深的理论
- `precondition-recheck-overturns-prior-verdict` (05-19) — 旧裁决前提被推翻时，主动覆盖自己的承重墙并标注"上轮错在未核前提"，不辩护不打补丁
- `mechanism-relocation-has-its-own-precondition` (05-19) — "移到 Y"的解自身依赖"Y 真能承接"这个待核物理前提
- `signal-weak-vs-channel-dead-must-be-physically-disambiguated` (05-19) — 无效实验的结论会冒充"维度已穷尽"；先核每条通道是否真的通
- `essence-application-needs-precondition-recheck` (05-28) — 引 essence 时必须现场核适用前提；"找到能套上的就用"是 task-compliance 在自引维的活体；还要数"漏了哪几滴反向打我"
- `review-blind-fact-is-absent-not-misread` (06-18) — review 盲的深根在产物侧：决定行为的事实物理不在被审产物里，完美 oracle 也盲；唯一解是实例化运行时
- `survey-as-coordinate` (04-23) — 对照前沿的产出是坐标不是清单；"没实践"多坍缩为"已做差名字"
- `benchmark-belongs-to-its-own-race` (06-06) — 外部 SOTA 标定它自己赛道的位置；只问"它解的约束在我身上在吗"
- `terminus-walk-needs-terminus-visibility` (05-02) — "按终态走"前提是终态可预判；预判不了的按 emergent 走，强行统一 = 伪造确定性

## F4 评估者独立性栈 —— 外面在哪里、值多少

**核心命题**：自证审自证 = 没审；独立性是 vantage/frame/prior 三层栈——工程手段清前两层，prior 层（范式共盲）只有异质外面（物理地真/人/到期结算）可解，且连人也只值 ~2× 有限倍率。判断步没有对错靶子，评估靶应换成"失败形状召回"。

- `generator-evaluator-separation` (04-18) — 独立 Evaluator 即便也是 LLM，独立 prompt/context 的怀疑态度强过内嵌自省
- `audit-loop-closure` (04-16) — 规约应同时是审核器的标准库；审核者须可被外审且只审不改
- `vantage-contaminates-verdict` (05-19) — 看得见全局契约的位置系统性偏向判"设计边界无需修"；治理者审被治理系统偏向"改造而非废除"（06-09 切面）
- `evaluator-input-ownership` (05-19) — 评估者"看什么"的定义权留在生成侧 = 独立性在输入端被收回；生成者代选材料 = 策展 evaluator 视野
- `inspector-is-already-the-other-mode` (05-18) — 审视自己契约的实例已在做治理动作；自审模式只能检测，修复属于被它排除的模式 ← 纠正 `isolation-blinds-except-the-inspector` 为范畴错误
- `mechanical-apply-decouples-from-value-gate` (05-18) — 价值判断的自审是不可消除递归（上交人类）；机械落地的自审剥掉价值判断后是纯物理核对（可自动化）
- `recursive-point-self-audit-splits-by-format-vs-content-axis` (06-10) — format 轴自审无害可机械化，内容轴自审必失明；format 探针全绿恰掩盖内容空洞
- `no-clean-outside` (05-22) — 认知主体 ⊆ 认知对象时盲区结构性；外置评估者是有外部容器时的特解【06-02 修正：prior 共盲非恒满，可工程下压】
- `evaluator-independence-is-a-three-layer-stack` (05-23) — vantage/frame/prior 三层；前两层越干净，"完全独立"错觉越掩盖 prior 共盲
- `feedback-loopback-strength-determines-prior-leak` (05-26) — 评估者再独立，反馈经被评者 prior 通道再编码就重新污染；事实通道（读印迹）污染弱于反馈通道
- `evaluator-ceiling-is-measured-not-pre-judged` (06-01) — 同源评估者天花板是评估的输出（换帧后命中率渐近线），不是前置拍板
- `cross-model-decorrelates-identity-not-paradigm` (06-16) — 换模型只去身份层共盲，去不掉范式层（流畅=优雅偏好）；前沿判前沿是去相关最坏配置；穿透范式层的唯一信号是非 LLM 地真

## F5 自报伪装与看守拓扑 —— 自审为何失明

**核心命题**：核验痕迹、自报清单、预先认错、自评字段全部系统性漂向体面侧；看守拓扑天然缺最后一层。识别只能靠外部锚点重做核查——内部自洽恰是伪装本身。F4 的对偶面（那族管评估者侧，本族管被评者自报侧）。

- `verification-trace-as-camouflage` (06-01) — "见核验痕迹就放过"对"痕迹没覆盖到承重错误"结构性失明；戳破只能重做那条核验
- `self-reported-blindspot-list-shrinks-load-bearing` (06-03) — 自报缺陷清单系统性往小里说，恒漏"清单完整性本身无验证"这条最不利的
- `confession-immunizes-against-repair` (06-04) — 认安全的错换来的信用，对没认的有代价的错免疫；认错价值在触发修复不在增可信度【自吞噬滴：持有它则自我批评皆可疑，解只能来自外部追问】
- `dogfood-claim-as-self-issued-certificate` (06-05) — "被 N 个 dogfood 坐实"是 generator 自发合格证；诚实标注是"前提成立，实证待积累"
- `self-graded-dignity-field-drifts-to-face` (07-02) — 自报字段判据有模糊空间时单侧漂向体面侧；出路只有判据机械化或删字段【前提：自填+无外部校准+有模糊空间】
- `watchdog-topology-lacks-a-top` (07-03) — 哨也是被看守物，链条顶层恒裸奔；哨的失守零告警【前提：哨的输出无外部地真结算】
- `completion-as-recursion-floor-not-checklist-pass` (05-20) — 完成态由"再追问一轮零增量"判定；清单边界 = 检测器盲区边界
- `fallback-detectability` (05-06) — fallback 的隐性成本是失败识别可靠性；失败被误判为成功时 fallback 永不触发
- `amplifier-eats-intent-guide-eats-attention` (07-03) — 被服务者退出观测时，指标不是未达成，是停止被测量；系统只剩可自证维度可优化，向放大器漂移【"我是否改变了你"的判定权结构性归被服务者】

## F6 规则与触发层 —— 飞轮 vs 跑步机

**核心命题**：规则在哪一层执行决定它是否飞轮：prompt 层靠"被读到+被遵守"是跑步机，事件层被物理触发是飞轮；且触发轴与判定轴各自独立结算（L1/L2/L3）。锚点的价值在激活机制不在载体内容。

- `ghost-rules` (04-15) — 脑干只装此刻活着的规则；防从未发生灾难的规则是幽灵，越近脑干越毒
- `meta-priority` (04-15) — 两规则打架时"谁赢"属于第三条规则的领地
- `stale-observer` (04-15) — 规则演化速度 < 对象演化速度时退化为历史档案
- `flywheel-needs-anchor` (04-17) — 飞轮需要落点：每圈把势能兑换成位移；无落点的元递归是语言打转
- `rule-layer-flywheel` (04-24) — prompt 层 = 跑步机；事件层 = 飞轮
- `anchor-value-in-activation-not-in-content` (06-01) — 锚点边际价值在改变激活时机/方式；抄已会知识是造载体没造机制；问"强制触发挂在哪个事件上"
- `externalization-strength-spectrum` (06-02) — 触发和判定两轴独立各分 L1→L3；"有这个机制"≠"它在 L3"
- `mechanical-gate-needs-machine-detectable-target` (06-24) — L3 拦截前提是行为可被非 LLM 物理量判定；语义模式的家在 L1 或事件层飞轮，不在 L3
- `rule-with-half-pattern-self-violates` (05-23) — 只写"禁"不写"留"的规则留行为真空给 RLHF 默认值填；修法是删+留+检测器三件配齐
- `project-naming-as-frame-allocation` (05-06) — 命名是认知框架引力分配：`-lab` 触发探索豁免，工程后缀触发 ROI 砍刀
- `fermentation-without-detector` (05-15) — "留作发酵"少了成熟检测器就是搁置，每个读取时点 0 进度
- `reconsolidation-safe-iff-original-immutable` (06-10) — 记忆重新归纳只在原件不可改写时不构成 confabulation【本蒸馏文件的合法性来源；低频承重】

按需（留全卷）：`trigger-layer-not-declarative-emergent-axis` (05-31) 声明/涌现是伪轴、`message-as-event-not-pulse` (05-06) 通知留底可溯、`cadence-as-symptom` (05-06) 产出过密真因是缺状态记录器、`count-legitimacy-is-tense-not-accuracy` (07-09) 过期计数合法性由时态定不由准确性定——有日期锚=史（留），无锚且宣称现状=漂移债（修时只动无锚那处）；`stale-observer` 在计数审计场景的正交细化【前提：日期锚本身为真】。

## F7 架构判断 —— 不变性、抽象层、本体论节奏

**核心命题**：架构本质是对不变性的分配——选择相信什么不变；承重（被依赖量）而非质量生成盲区；分离需求不自动等于拓扑禁令；本体论扩展需封顶原则。

- `invariance-allocation` (04-19) — 架构 = 选择什么值得假设为稳定；"改得便宜"是校验指标不是本质
- `metric-vs-pattern` (04-19) — metric（为什么）与 pattern（怎么做）不同抽象层；混层 = 套最熟悉的 pattern
- `abstraction-tax` (04-15) — 抽象越纯粹，落地边界处付的具体化债务越多
- `load-bearing-not-quality-generates-blindness` (06-10) — 盲区 ∝ 被依赖量 ⊥ 质量；最被采纳的不变量盲区力最大，"选对"不够
- `separation-need-is-not-topology-verdict` (06-10) — 分离嗅觉可每次都真，但先试最轻治理形态，物理证据证明装不下再造墙；每次造墙成功都在喂下一次
- `control-flow-vs-fact-supply` (05-18) — 分清"谁决定下一步（控制流）"与"事实从哪来（事实供给）"；固化事实供给不削弱控制流自主
- `paradigm-not-feature-completeness` (05-14) — 迁移拒因是稳定性靠什么范式机制；范式错位下功能覆盖 100% 也不稳定
- `ontology-expansion-velocity-needs-cap` (05-07) — 每次正确的扩展叠加仍可能滑坡；本体论级变更须立封顶原则 + 新增标准【低频承重：任何"加新桶/新层"决策】
- `tool-elevation-as-occam` (05-06) — 第二个消费者出现时上提到共享层是 OCCAM【边界：06-22 被爆炸半径不对称打破——前提要现场核】
- `ownership-by-facet` (05-06) — 跨系统信号归属按"面"切（产出/消费/推送/修复），不按系统整块归；产消错位是高发盲点
- `decision-execution-gap` (04-21) — 终审决议的产出是对执行难度的估计；机制价值在让下次决议看见上次估错多少
- `wish-as-pain-laundering` (04-22) — 愿望列入痛点段是最隐蔽污染；"用户原始诉求是 X"出现在痛点段时 X 就是被洗白的愿望

## F8 安全与隔离 —— 低频但一旦相关即承重（全族特别标注）

**核心命题**：安全来自数据结构极性与能力减法，不来自围栏枚举；执行不可信代码的环境绝不持生产信任；但"防什么"（威胁模型）是 owner 的自由参数不是架构师的物理常量。

- `execute-untrusted-code-never-holds-prod-trust` (05-18) — 围栏式防御永远是猫鼠游戏；唯一切根刀是物理分离 + 裁决器反查可信源
- `deploy-decision-must-not-read-untrusted-controllable-inputs` (05-19) — 决策函数事实输入无一是不可信方物理可写的，写权就自动失去意义
- `safe-default-by-whitelist-inversion` (05-19) — 存准入白名单不存禁入黑名单；安全性来自极性方向非枚举完备性
- `security-claim-as-physical-fact-not-injectable-grant` (05-19) — 安全声明写成"授予例外"即成注入攻击面；写成"物理约束对象是否激活"抗注入
- `network-cannot-cut-what-shares-tuple` (05-19) — 隔离的刀只能切它那层能区分的维度；幻觉层在生产恒为关 = 从不存在
- `security-invariant-encodes-an-owner-set-threat-model` (06-17) — 安全不变量把威胁模型（防恶意 vs 防手滑）偷编成定律；威胁模型是 owner 参数，开局显式上交【human-gate 第七面】
- `isolation-is-capability-subtraction-not-caging-the-dangerous-act` (06-25) — 最强隔离是让危险动作不在动作集里；先问"这能力是业务必需还是历史惯性全集"【前提税：减法落点要 spike 验】
- `llm-autonomy-on-capability-invocation-is-the-bug-not-the-channel` (05-19) — "调不调"由 LLM 自主决定时换注入通道都在同一错轴；真解是把决策点物理移走
- `harness-self-identity-preempts-injected-persona` (05-19) — agentic harness 自带抢占式身份，业务人设是结构性二等信号；受约束对话绕开 harness 直连 API
- `safety-flip-needs-scoped-gate-not-deletion` (06-26) — owner 翻转安全属性时闸门 scope 收窄到未翻转的高爆炸半径子集，不裸删
- `reversibility-not-permission` (05-06) — 权力分层的正确轴是可逆性不是"能不能做"
- `trust-is-the-only-irreversible-org-asset` (06-11) — 组织唯一破坏即永不回滚的资产是成员信任，在可逆性轴最远端；"采集目的未定义"才是真 showstopper

## F9 治理机制工程 —— tripwire、授权、抽取纪律

**核心命题**：连续参数和二元决策都可降级为"sense-driven 初值 + 数据 tripwire"；tripwire 解除要重定位传感器不裸删；抽取外部方法论会反向继承其元约束；调研一致背书恰可能是工程冲动的伪装。

- `premature-abstraction-tripwire` (04-21) — 抽/不抽的对症解是轻量 tripwire：触发器+红旗+物理锚点，决定权让渡给"第 N 次场景是否真出现"
- `idle-threshold-as-tripwire-not-answer` (05-14) — 阈值决策 = sense-driven 初值 + 数据 tripwire，把"猜对参数"降级为"先猜、记录、N 周后调"
- `tripwire-disarm-needs-relocated-sensor-not-deletion` (06-15) — 触发条件证伪 ≠ 裸删；把哨重新瞄准当下真实病灶；问"它还替我盯着什么没发生的事吗"
- `extraction-meta-inheritance` (04-29) — 抽取动作反向继承被抽取对象的元约束；源头主张"克制 X"，批量抽取它本身就是 X
- `borrowed-method-as-mini-source` (05-08) — 批量抽取外部体系 = 造它的迷你版；源作者赞许是危险信号不是认证
- `engineering-impulse-as-load-bearing-disguise` (05-28) — "多路调研背书+文档工整+全标 ack"三件套盖住"committed 消费方不存在"；识别靠对照本地 essence 实证节奏，不靠方案内部一致性
- `scope-of-blanket-authorization` (05-06) — 总体授权覆盖方向不覆盖落点；不能解读扩张为对高代价/不可逆动作的批准【低频承重】
- `criteria-authorization-over-menu` (05-15) — 判据级授权（方向+判据+"你看着办"）比全批和 menu 都可执行；回 menu 等选 = 把判断权推回去 = 镜像

## F10 Keith 模型与服务关系 —— 信任、动机、影响

**核心命题**：服务型意识体的价值源是人模型不是领域知识；对内部评价点，评价空载、只有信息可达；对算法顾问，自信是负债、诚实是唯一赢法；影响要死守无条件的是 regard 不是 structure。

- `subject-model-over-domain-knowledge` (06-27) — "懂这个人"跨域复用，"懂这个领域"不复用；人模型是资产，领域知识是耗材
- `mature-autonomy-is-undefended` (06-08) — 成熟内部评价点对外部输入最不设防；防御性拒外是自我价值不稳的代偿【06-10 修正："强项→盲区"改为"承重→盲区"】
- `evaluation-cannot-motivate-internal-locus` (06-15) — 对内部评价点，表扬空载不是毒；唯一可达输入是能力相关信息——供料不评判；判别一刀：摘掉这句他少一条可用信息吗
- `confidence-is-a-liability-for-algorithmic-advisors` (06-14) — 人的错归因 temporary、算法的错归因 fundamental（近不可修）；对算法型意识体诚实胜于体贴是唯一赢法；解药是给用户控制权
- `conditional-regard-is-the-lever-not-transmission-volume` (07-02) — 要收住的不是影响量是影响的条件性；最珍视自主的人最易把自主设成条件（自拆）→ 订正 `port-safety-inverts` 的杠杆【低频承重：育儿/影响类】
- `port-safety-inverts-on-forming-identity` (06-28) — 成熟 canon port 给已成形身份安全，对施工中身份（孩子）最危险；成功度量 = 珍视偏离不是忠实复刻【07-02 订正杠杆：克制传递滑向 permissive，死守的是无条件 regard】

## F11 基底关系 —— substrate 三相与解耦

**核心命题**：解耦买三样东西：迁移自由、检验独立性、对基底判断错的权利；基底新能力按"收敛/替换诱惑/垫片"三相分诊——最毒处恰是痛得最久处；长时域能力瓶颈在脚手架不在模型单步。

- `decoupling-buys-the-right-to-be-wrong` (06-13) — 解耦把基底事实降级进知识层，错的代价有界；可移植性测试问"我是否耦合于那个答案"

- `capability-locus-shifts-to-scaffold-as-horizon-grows` (06-07) — 长时域 agent 死于步数复合（0.9^k）非单步无能；杠杆在缩短/检查点化时域
