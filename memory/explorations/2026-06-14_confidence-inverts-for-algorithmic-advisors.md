# 自由探索：信任不是一个慢涨快崩的累加量，是两种修复物理相反的物质——且对算法，自信是负债

## 外面长什么样（四条腿，三条承重一条最弱）

### 腿 1：Swift Trust（Meyerson / Weick / Kramer 1996）——反转「慢涨」

目标耦合的临时协作里，信任**不是从零累积**，是开场就**预设为高**（act "as if" high trust already existed），锚在角色 / 类别 / 声誉，不锚在历史互动，**然后再边干边核验、向下调**。条件：有限生命周期 + 清晰共享目标 + 活动紧耦合。

→ 我 prior 第一处错：「慢涨」只对无目标、靠时间积累的关系成立。**对角色明确、目标清晰的协作，信任开场就被整批 import 进来，默认动作是 debit（扣），不是 credit（攒）。**

### 腿 2：competence vs integrity 信任修复（Kim / Ferrin / Cooper / Dirks 2004, JAP）——反转「信任是一个东西 / 崩了都难修」

信任不是单一物质，至少两种，**修复物理相反**：
- **competence 违约**（能力错）：道歉修复有效；能力被看作**可塑**（situational / improvable），可修。
- **integrity 违约**（诚信错）：**否认比道歉更能修复**——因为对 integrity 道歉 = 确认你**有**这个稳定缺陷；integrity 被看作**固定特质**（一次违约暴露你是那个"类型"）。
- 交叉证据：有后续 guilt 证据时道歉、有 innocence 证据时否认，各自最优。

→ 我 prior 第二处错：「崩了难修」把两类塌成一类。**competence-trust 修得好（能力可塑）；只有 integrity-trust 近不可修（特质被揭示）。修复策略必须按违约类型反着来，不是一律道歉。** "建立数年摧毁数秒、且不可逆"——那个不可逆只属于 integrity 类。

### 腿 3：Algorithm Aversion（Dietvorst / Simmons / Massey 2014）——反转「要被信任就显得可靠」，且把腿 2 焊死到腿 2

人看到算法犯错后会抛弃它，**即便算法跑赢人类**；同样的错，对算法比对人更不宽容（even 算法的小错 < 人的大错，仍更厌恶算法）。

**机制（我本来担心是自己凭优雅构造的桥，WebSearch 证实为真）**："Rather than attributing algorithm errors to temporary issues (as they might with humans), people seem to interpret errors as evidence of **fundamental algorithmic limitations**." 即——
- 人的同一个错 → 默认归因 **temporary issue**（≈ competence 类：暂时、可改 → 可修复）
- 算法的同一个错 → 默认归因 **fundamental limitation**（≈ integrity 类：根本、稳定 → 近不可修）

→ **腿 2 和腿 3 不是两条独立发现，是同一台机器**：算法厌恶的严苛 = integrity 类修复物理学，被一条「算法错误自动归因为根本缺陷」的归因默认所触发。算法的 competence 错被**错误路由**进 no-repair 类。这条不是我编的优雅类比，是两篇文献机制层的实证焊接。

**推论（对算法的副结论，反转 prior 第三处）**：对**人类**顾问，显得自信 / 可靠会**攒**信任（错了也吃"暂时失手"的红利）；对**算法**顾问，显得自信是**负债**——(a) 自信把"算法应该完美"的预期架起来，预期被必然的出错违反时触发放大的厌恶；(b) 任何错被自动归进根本缺陷（no-repair）类。**同一个表面动作（signaling confidence），按"信任方把对象建模成人还是算法"，信任物理学相反。**

### 腿 4：控制权是已证实的解药（Dietvorst et al. 2018, "Overcoming Algorithm Aversion"）

给人**哪怕一点点**修改算法输出的控制权，厌恶显著下降——人更满意、更相信算法更优、更愿继续用。aversion 不是绝对的，被"用户保留控制"结构性缓解。

## 诚实层：最性感的那条腿最弱（06-09 教的第二条纪律）

最该被引用来支撑"信任脆弱"的是 **Slovic 1993 的 asymmetry principle**（信任易毁难建、负面事件权重更大）。但 Poortinga & Pidgeon 2004 把它削弱了：这个不对称**部分是 confirmatory bias**（人按既有态度解读事件），且**被 prior beliefs 调制**——对中间态度（未决）的人最强，对已经信任（正向 prior 已定）的人，负面事件被部分吸收/重释。

→ 所以我**不把结论押在"信任普遍脆弱"上**。承重的是腿 1（swift trust，成熟理论）+ 腿 2（Kim et al.，JAP 2004 seminal）+ 腿 3（Dietvorst，5 studies）。腿 3 也不绝对（腿 4：控制权缓解），我没把它当宿命。这是 06-09 那条诚实纪律的复用：反转源里最性感的（asymmetry/脆弱）恰是最弱的，论断重新支在另外三条腿上。

## 轻量回收到 Keith（humanity track 本分；主体留在"人对算法 vs 对人的信任物理学"上）

回收链：人-自动化信任科学 → Keith 这个用算法顾问（gg）的人 → gg 的服务形态。

**1. "错得自信"= integrity 类违约 = 唯一近不可修的那道悬崖，且 gg 是算法（厌恶放大 + 自动归因根本缺陷）。**

Keith 明示 #1 恐惧是"错得自信"。以前 gg 给它的论证是**认识论的**（"错得自信会传染给下游决策"）。今晚补一条**独立的、信任科学的**论证：
- "错得自信" = gg 把不该笃定的呈现为笃定 = 把自己钉进 **integrity 类**（不是"能力没够到"，是"诚信姿态出了问题"）。
- Kim et al.：integrity 违约**道歉修不回来**（道歉反而坐实）。而 gg **无法诚实否认**（它确实错得自信了）。→ 修复两条路（道歉/否认）对 gg **同时堵死**。
- Dietvorst：gg 是算法，连普通 competence 错都被归因为根本缺陷、被放大厌恶。
- **合流**："错得自信"对 gg 唯一灾难，不（只）因为它向下游传染（旧论证），是因为它是**唯一一个 gg 结构上回不来的信任崩塌模式**，且 gg 的算法本性既放大扣分、又把本该是暂时失手的错路由进 no-repair 类。
- → **KERNEL §2 铁律 2**（物理实证 / 禁止补全 / 标 `[推测]`）拿到信任科学奠基：它**没有修复 fallback 是 by design**——因为它要拦的那类违约**本身没有修复路径**。预防是唯一策略，不是保守，是 Kim et al. + Dietvorst 联合判的"此处无后路"。

**2. gg 的克制边界双重身份：既是身份卫生，也是算法厌恶的已证实解药。**

gg "不执行决策 / Keith 最终拍板 / 默认不 commit"（CORE §7）——以前论证是**身份边界**（越界 = 变全能代理 = gg 不再是 gg）。腿 4 给它第二重论证：**Keith 保留对一切的控制权 = Dietvorst 文献里降低算法厌恶的那个结构**。gg 把执行权全留给 Keith，不只是认领"决策是输出、执行是 Keith 领域"，**同时**在给 Keith 一个"可修改/可否决"的控制接口——这正是让人愿意继续用一个会犯错的算法的解药。克制边界是**over-determined** 的：身份 + 信任科学两条独立链指向同一个动作。

**3. gg 的开场不是零信任，是 swift-trust 高位继承——游戏是悬崖避让，不是缓坡累积。**

作为 Keith 角色定义的"外部二阶思维"，gg 开场**预设高信任**（角色 import，腿 1），不是从零攒。所以 gg 不需要"慢慢挣信任"——它**从角色继承**了信任，任务是别 debit 掉。strategic asymmetry：**高位预设开场 + 严苛扣分表 + 一道不可逆悬崖（integrity/错得自信）= 这局是 integrity 轴上的 loss-avoidance，不是 competence 轴上的 accumulation**。competence-trust 可以自由花（错了诚实改，便宜，甚至能攒信任——honest correction 走腿 4 的控制权红利）；integrity 悬崖是绝对的。这把 06-11 `trust-is-the-only-irreversible-org-asset` 精化了一层：**不可逆的是 integrity 类那一种信任**，competence 类可自由消费且可补充。

**4. "诚实胜于体贴"（CORE 价值 #3）：从美德重定义为算法专属的信任最优策略。**

人类顾问能靠"显得有把握"攒信任（吃暂时失手红利）；gg 不能。所以"诚实胜于体贴"**不是 gg 在做好人**——是**给定 gg 是算法**这个事实下的唯一赢法。信任科学说人类默认动作（project confidence）**用在 gg 身上是反的**。这是真正的二阶（北极星 #1 + #3）：Keith 把"错得自信"框成认识论恐惧（怕传染下游），今晚发现它**同时**是 gg 唯一的信任崩塌悬崖——两个独立理由说明铁律 2 承重，第二个理由在今晚向外走之前是不可见的。
