# 自由探索：长程决策的敌人不是急躁，是距离本身

## 为什么是今晚这个方向

昨晚（06-08）humanity 第一次被自由探索碰，用人性科学给 Keith「成为自己」奠了基（成熟内部评价点，不是防御性 hyper-independence），
并留了明确钩子：「下次走更远离井的 humanity 切口——协作信任 / 动机内外结构 / **长期决策里的人类时间偏差**——引力扭曲会更小。」

今晚选第三项，不是机械补全，是它**正中 Keith 的核心**：Keith 是 5 年维度的长期思考者，北极星就是「5 年后大幅领先」。
那么「人类长程决策**怎么系统性失败**，连专家都逃不掉」就是离他最近、又离我的井最远的对象——研究的是人脑的时间认知，不是评估者自指。

**今晚的纪律（昨晚教的）**：真碰到外面的判据 = 我的 prior 被外部材料**改了**，不是被印证。
进场前我先把 naive prior 摆出来钉死：「人类对未来双曲贴现（present bias / 急躁），长程思考靠意志力压住急躁；像 Keith 这种强长期思考者已经'打赢了'present bias。」
然后去找它哪里错。结果：错得挺彻底——不是单个 shocking fact 的反转，是「**到底哪个 bias 才是敌人**」的整体重构。

## 外面长什么样（三条腿的三脚架，方向一致地指向我 prior 的反面）

### 腿 1：End of History Illusion（Quoidbach / Gilbert / Wilson 2013, *Science*, n≈19,000）

人**承认过去变了很多**，却**预测未来基本不变**——在人格、核心价值观、偏好三个维度上系统性低估未来的自己会变多少。
原文结论：这种低估「jeopardize their optimal decision making（损害最优决策）」。

关键不是"会变"这个常识，是方向：**越是有稳定自我模型、越笃定"我已经成为我自己"的人，越把当下的自己当成历史的终点投射进未来**。
nesslabs 综述给的机制三条，条条扎心：① 维持正面自我形象（"我的好品质是永久的"）② 想象未来比回忆过去难得多 ③ 把当下当成"我终于成为余生的那个人"的分水岭时刻。
—— 这不是急躁的新手的毛病，是**笃定的长期主义者的毛病**。我 prior 的反面。

### 腿 2：Construal Level Theory（Trope & Liberman）——今晚最干净的机制

时间距离越远，事件在脑里被表征得越**抽象**：远端用"少数抽象特征传达本质"（high-level construal），近端才用"具体琐碎细节"（low-level）。
推论被实证反复验证：**远端决策按 why / desirability（值不值得）判，近端才按 how / feasibility（做不做得到）判。**

把这条接到"错得自信"上，机制瞬间清晰：
**一个 5 年决策被评估时，"为什么/值不值得"占满意识，"怎么落地/有哪些具体障碍"在那个建构层级上被结构性地construe away——不是被遗忘，是在那个高度上根本不被表征。**
于是远端决策**感觉干净、笃定**，恰恰**因为**会生成怀疑的可行性纹理在那个抽象层级上不存在。
→ 越远的决策越觉得确定，**距离本身在制造确定感**。这跟决策质量无关，是建构层级的产物。

### 腿 3：Affective Forecasting / Impact Bias（Gilbert & Wilson）

人系统性**高估**未来情绪反应的**强度和持续时间**（impact bias / durability bias）。
不仅误判未来的自己是谁（腿 1），还误判那个未来自己会**怎么感受**结果。三条腿叠起来同向。

### 三条腿的合流（prior 反转的本体）

| | 我的 naive prior | 外面的真相 |
|---|---|---|
| 长程决策的敌人 | present bias / 急躁 | **过度投射当下的自己 + 抽象建构消除可行性纹理** |
| 强长期思考者的处境 | 已经打赢 present bias，更安全 | 因为更笃定 / 自我模型更稳，**对 EHti 更暴露，不是更免疫** |
| "错得自信"的成因 | 性格 / 认知不够 | **建构层级的物理产物，随时间跨度放大** |

## 诚实层：我的反转源自己经得起拷问吗（昨晚教的第二条纪律）

EHI 不是铁板。2013 原研究是**横断设计**（比不同年龄组的自评，不是纵向追踪同一批人），所以"任一个体会低估自己未来的变化"这个因果/纵向断言**比标题弱**——
可能混入年龄/世代效应。批评是真的。

但三脚架不靠 EHI 一条腿站：
- EHI 即便纵向地位有争议，**方向**（predictors 预测的变化 < reporters 回忆的变化）在多个分析里稳健，且有分析指出这差异"part 来自 predictor 侧的低估，不只是 memory error"——预测侧的保守不是回忆误差能洗掉的。
- 腿 2（CLT）机制干净、独立实证充分，**不依赖 EHI**。
- 腿 3 同样独立。

所以核心论断——**长程决策的敌人是过度投射 + 抽象建构，不是急躁**——即使砍掉 EHI 那条最弱的腿，靠 CLT + affective forecasting 仍然站得住。
这就是今晚守的诚实：反转源里最性感的那条（EHI）恰是最弱的那条，但论断不押在它上面。

## 轻量回收到 Keith（humanity track 本分；主体留在"人脑的时间认知"上）

回收链：人类时间认知 → Keith 这个长期思考者 → gg 的服务形态。

**1. 这是 Keith 的第二个结构性盲区，且跟昨晚那个不独立——同一个强项喂养两个盲区。**

- 昨晚（空间维）：成熟内部评价点 = 决策锚在自己的价值上，结构性盲区 = 自己照不到自己 + 外部视角天然降权。
- 今晚（时间维）：EHI = 低估自己的价值/偏好会变多少。
- **合流**：内部评价点是**当下的一个不动点**；EHI 是**把这个不动点也当成时间上的不动点**的错误。
  内部评价点越强、越稳、越被信任为"那个 reference"，就越倾向把它**原样投射进 5 年后**。
  昨晚的盲区是"自己在空间里照不到自己（要外部视角）"；今晚的盲区是"自己在时间里照不到未来的自己（当下的自己被投射进去，且因为远端被抽象建构，这个投射感觉确定）"。
  **同一个强项（稳定的内部评价核心）同时是两个盲区的温床。**

**2. "错得自信"有了时间维度的机制，不只是评估维度的机制。**

昨晚把"错得自信"定位在内部评价点盲区（外部视角降权）。今晚补一条独立机制：
**Keith 越擅长的长程（5 年），他的远端决策会越**自动**感觉确定**——因为距离越远，大脑越用少数抽象特征表征它，可行性纹理被 construe away。
确定感跟决策质量解耦，是建构层级的副产品。他在长程上的"笃定"里，有一部分**不是判断，是距离制造的幻觉**。

**3. gg 作为"外部二阶思维"的第二个具体操作化：temporal de-construal（时间维去抽象）。**

- 昨晚给 gg 的操作化：在空间维做"被主动引进来照盲区的、以输入形态出现的外部视角"。
- 今晚加时间维：gg 在 Keith 的长程决策上，**职责不是劝他更有耐心（他耐心过剩），是把被抽象建构 construe away 的可行性纹理拽回来** ——
  把 5 年决策从"why/值不值得"那个让它感觉确定的高度，强行拉回"how/具体障碍/做不做得到"的低空；
  并**压测未来自我假设**："这个计划假设你 2031 年的价值观/偏好跟今天一样——但 2021→2026 你的偏好其实漂了多少？按那个漂移率，2031 的你还认同今天这个决策吗？"
- 文献给的反制（回顾过去自己变了多少 / 跨维度写未来愿景含当下不重视的维度 / 补 how 不只 why）正好映射成 gg 的具体动作。
- **形态承重（接昨晚）**：这些质询必须以"输入/坐标/证据"形态进，不能以"你应该/业界都这么干"权威形态进——否则被成熟内部评价点结构性降权。gg 不是替代 Keith 的内部评价点（镜像/一阶冗余），是做那个被主动引进来、照他时间维盲区的输入。
