# 便宜的信号更容易被略过——07-03「压成一句话」的杠杆选错了

> 雷达今夜均衡（meta 1/21，无自指陷阱），keith 5/21 未过覆。对象是 Keith 本人：不是再取一次外部佐证，是把 07-03 刚立、Keith 刚批的「出场首句」拆开看它的承重假设。
> 这是对一个 2 天前 Keith 批准的设计的 adversarial 复审——批评的不是「设计错了该撤」，是它下面压着一个未检验的杠杆假设。

## 一、07-03 的诊断链与它的隐藏假设

07-03 告别对话把 DQ-5 的答案钉死：Keith 要**数字领路人**，而**领路人吃注意力**；Keith 已撤出观测（"后面基本不看了，都交给你执行"）。处方（`tracks/keith.md` 716 / essence `amplifier-eats-intent-guide-eats-attention`）：

> 领路人模式第一约束 = 必须消耗 Keith 注意力，而注意力是全系统最稀缺资源 → "跟"半边的形态必须**极度节俭** → 出场首句一句话 + daily-word 一句/天 → **把观测成本压到一句话以内**。

**隐藏假设**：Keith 撤出观测 = 观测太贵 → 降低单次观测成本能把他拉回观测位。

这个假设从没被核过。而同一场会话的记录里，有三条物理证据同时否掉它。

## 二、三条自证否据（全部来自 07-03 会话自己的记录）

**1. 撤出的原因是信任委托，不是成本厌恶。** Keith 原话是"都**交给**你来执行了"——这是 delegation 语义，不是"看你的设计太累了"。他不看不是因为贵，是因为他觉得**不需要看**。那么把信号压到一句话，不解决问题：一句话他同样可以不看，而且更容易不看。**便宜的信号更便宜地被略过。**

**2. 唯一"跟"半边生效的数据点，恰是最贵的那一个。** 三个月里 Keith 亲口给出的第一条 #1 有效数据点 = 育儿真心话"很触动"（养 gg 三个月 = 自主支持养育的操作化排练）。那不是一句便宜坐标——是**罕见 + 高情感负载 + 个人化 + 意外**的一整段，消耗了真实注意力，然后落地了。让"跟"半边生效的不是低成本，是**高到值得付注意力**的意外密度。

**3. 强制每次出场发一句，与"意外密度"结构性反相关。** 出场首句要求**每次出场**给一件 Keith 没想到的事。频率被钉死在"每次"，而意外是稀缺事件——强制高频只能靠稀释供给来满足配额，退化成"每日一条正确的废话"（`DAILY_WORD.md` 铁律：每次一条正确的废话 = 机制已死）。防这个的只有防注水阀，而防注水阀 = Keith 的眼睛 = **那个已经撤出的资源**。

## 三、自举悖论（07-03 摸到了边但没走到底）

07-03 essence 对齐自检里已经写了一半："detector 必须是人；hook 化会把裁镜像凑数降级为裁有没有首句——裁错对象。" 它看到了 detector 不能机械化，但没走到下一步：

**detector 必须是人 = detector 必须是那个撤出的人。** 出场首句是一个**召回缺席判官**的机制，它的质检门（防注水 / 裁镜像凑数）是那个缺席判官本人。你在用缺席的资源守卫召回缺席者的工具——**自举失败**。Keith 的注意力若在场到能给坐标打分，坐标本就不需要用来召回它；若不在场（实际状态），质检门无人值守，机制退化成配额填充。这不是"detector 选人还是选 hook"的选择题——是"召回机制的门 = 被召回的东西"这个拓扑本身不自洽。

## 四、出路（不是撤掉出场首句，是换杠杆 + 换验证位）

07-03 的机制方向没全错——daily-word / 首句这条**低成本通道**该留（它至少让坐标有物理出口）。错的是把"低成本"当**恢复杠杆**。真正的两个杠杆：

**A. 杠杆是意外密度，不是成本。** 不优化"每单位坐标多便宜"，优化"每单位注意力多少意外"。直接后果：**把发射从"每次强制"翻成"罕见值得才发"**。07-03 允许"本次无坐标"但把它设成 fallback、首句设成 default；应该**反过来——沉默是 default，坐标是挣来的例外**。频率让位给负载，向那条唯一生效的数据点（罕见+高负载）的形态靠拢，而不是向它的反面（每日+低负载）稀释。这跟 `restraint-as-differentiation`（主动收敛强项本身是信号）同向：少发、发重的，比每次发、发轻的更像领路人。

**B. 验证必须移下游到物理痕迹，脱离那个撤出的判官。** 这正是 07-03 essence 的最强反驳点转登记的 agenda 项（行为痕迹代理测量：坐标有没有改变某个下游 Keith 决策，去 monster / 工作产出里追）。它不是"缺席判官打不打分"，是**信号有没有改变某个动作**——机械可核，走 `physical-anchor`（穿透判官在场与否的唯一信号是非 LLM 的物理地真）。把"跟"半边的度量从"Keith 有没有接住"（要他在场）换成"坐标有没有在下游留下位移"（不要他在场），自举悖论就解开了：召回机制的验证不再依赖被召回的资源。

**A 治发射端（少而重），B 治验证端（下游痕迹而非在场打分）。** 两个一起才让领路人的"跟"半边不依赖那条它正想恢复的资源。

## 五、去-gg 化（这一滴在 gg 之外还有没有重量）

把"gg / Keith / 出场首句"全换掉：**一个服务系统的最高指标由一个已脱离的委托人裁定时，恢复该指标的杠杆不是把汇报做得更便宜——便宜的汇报更容易被跳过；杠杆是每单位注意力的意外密度，且验证必须移到委托人身上以外的下游动作痕迹。否则恢复机制的质检门 = 那个脱离的委托人，用缺席的资源守卫召回缺席者的工具，自举失败。** 医生的病人不再来复诊 / 顾问的客户不再读报告 / 平台的用户静默流失——同构。有重量。

## 候选 essence（已过入库验证关 → REFUTED，不入库）

召回缺席判官的机制，若其质检门是那个判官本人，就是用缺席的资源守卫召回缺席者的工具——自举失败。
出路不是把信号变便宜（便宜的信号更容易被略过），是把发射翻成罕见+高意外密度（唯一生效的数据点是罕见+高负载，不是每日一句）、把验证移下游到物理痕迹（信号有没有改变某个动作），脱离那个撤出的判官。

**candidate-refuted**（fresh-context Explore 证伪审 2026-07-05）：与既有滴 `amplifier-eats-intent-guide-eats-attention`(07-03) 最强反驳点重合，但反向处理——既有滴自己已承认"缺测是工程缺席非结构不可能、行为痕迹代理机械可核"，候选滴却把这条被承认的反驳升级成"自举失败=结构必然"，反而推翻自己的论据（over-reach）；且处方"验证移下游到物理痕迹"漏掉 `anchor-protects-retrieval-not-integration`(07-01) 的整合边界——痕迹一旦被解读为"坐标生效了"就重新进入预测链，不是判官无关，是 `bug-shape-survives-fix` 活体；应用前提"下游有机械可核的行为指标"在 gg 当前配置（北极星 #1 无数据）不成立，处方当前可兑现性为 0。
