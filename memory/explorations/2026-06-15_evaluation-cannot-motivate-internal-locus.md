# 自由探索：对内部评价点的心智，外部评价是空载不是毒——能到达动机的只有信息

## 为什么是今晚这个方向

humanity track 第四晚。前三晚的结构：06-08 空间维（成熟内部评价点，自己照不到自己）/ 06-09 时间维（距离制造确定感）都是 intrapersonal；06-14 人际维（人怎么信一个非人协作者）是 interpersonal 的**信任**机器。06-14 末尾留了两个钩子，并显式判定哪个更远：

> 下次更远的 humanity 切口：动机的内在/外在结构（SDT 的另一半，跟评估者井零重叠），或协作信任**崩塌后**的重建动力学（今晚只碰了修复策略，没碰重建轨迹）。

后者是同一台信任机器再往前看一格（近）；前者换一台完全不同的机器（**动机**），且跟「gg 能不能信自己判断」那个 meta 井零重叠。选远的。

06-08 用了 SDT 的一半（autonomy/relatedness：成熟自主对外部输入反而最不设防）。今晚补**另一半**——动机的内在/外在结构（intrinsic/extrinsic + overjustification + 评价的形态）。问题钉死成一句：

> **怎么"激励"一个内部评价点、"只成为自己"的人？gg 该用什么形态给他反馈——表扬/肯定有用吗？**

## 进场前钉死的 naive prior（06-09 纪律：先钉死，再去找它哪里错）

> 动机 = 想要结果。激励一个人靠奖励好工作、表扬进步、肯定成绩、鼓励。认可越多→动机越强。所以 gg 应该肯定 Keith 的好决策、为他的胜利喝彩、给正反馈让他保持势头。

去找它哪里错。结果：**错了两次**——naive prior 错，我用来纠它的那个"优雅反转"也错。真相在第三个位置。

## 第一道引力（我差点直接写的优雅反转）——被文献当场证伪

我的第一直觉反转，干净又对称：Keith 是自主取向（autonomy orientation），**外部表扬会被他体验为控制（controlling）→ crowding-out → 表扬反而扑灭内在动机**。跟 naive prior 完美镜像：naive 说"表扬=燃料"，反转说"表扬=毒"。

去查 General Causality Orientations 对 reward 的调制——文献给的是**反方向**：

> Cantarero & van Tilburg：causality orientation **调制** reward 的 undermining 效应。金钱奖励扑灭内在动机的经典效应**只在 control-oriented 受试者身上复现**；autonomy-oriented 受试者在有奖/无奖两种条件下内在动机**一样高**——自主取向**屏蔽**了 crowding-out。

→ **自主取向不是让人更容易被表扬扑灭，是让人被屏蔽。** 我那个性感的对称反转（autonomy→表扬即控制→中毒）被精确证伪。这是 `elegance-is-refutation-resistance-not-truth` 第二次在 humanity 夜逮到我：最性感的那条桥（"对他一切评价都反噬"）是最弱的，文献主动反对它。`bug-shape-survives-fix` 活体——06-09/06-14 刚教过"最性感的腿最弱"，今晚第一直觉仍奔向最性感的腿。

## 外面真正长什么样（四条腿，承重的是 1/3/4）

### 腿 1：CET——评价的形态决定符号（informational vs controlling）

Cognitive Evaluation Theory（Deci & Ryan，SDT 子理论）：外部事件有两面——
- **informational**（信息面）：支持内部因果点 + 胜任感 → **增**内在动机
- **controlling**（控制面）：把因果点推向外部、施加压力 → **减**内在动机

关键：**口头表扬本身一般是 informational 的、倾向增内在动机——除非以控制形态递送。** "positive feedback 的收益在它带 evaluative 而非 informational 意味时被削弱。"

### 腿 2：自主取向屏蔽 crowding-out（用来证伪反转，不承重）

见上。Keith 是自主取向（06-08 成熟内部评价点 / `self-as-only-reference`「成为自己·没有偶像」/ 拒绝讨好 / 能力>体验 / 第一性原理），故**结构上被屏蔽**：表扬扑不灭他的内在动机。我**只拿这条证伪我的反转，不在它上面压重**（单一谱系、不绝对）——但它足以拆掉"表扬=毒"的地基。

### 腿 3：person vs process 表扬（Henderlong & Lepper 2002 综述；Mueller & Dweck；Haimovitz & Corpus 2011）——承重

- **person 表扬**（"你真聪明 / 你真厉害"，夸全局属性）：**减**内在动机，失败后诱发 helpless 反应（自责），教人对"全局价值"做推断。
- **process 表扬**（具体、关于策略/努力/**可控因素**）：**增**内在动机 + 胜任感。
- 一句话：表扬指向**可控力**（策略、做法）才增动机；指向**人本身**则减。

### 腿 4：控制型措辞即便是正向也伤（autonomy-supportive vs controlling language）——承重

> controlling（相对 autonomy-supportive）的反馈**即便内容是正向的也压低内在动机**（例："Good, you're doing as you should"）。控制型语言靠 pressure 措辞（"should""have to"）；autonomy-supportive 靠 noncontrolling + informational 措辞。

→ "你做对了，应该的"这种正向但**评判+应然**的措辞，仍是控制面主导。

## 把三条腿合成（naive 错、反转也错、真相在第三处）

对一个**内部评价点**的心智，表扬对动机的净效应**近零、不是负**（屏蔽住了），但它作为**信号**按形态劈成两半：

| 形态 | 对 Keith 动机 | 作为信号 | 净判 |
|---|---|---|---|
| **person/evaluative 表扬**（"好决策""漂亮""厉害"） | ≈0（屏蔽，不扑灭） | 无胜任内容可用 + 带 evaluative 意味，被成熟内部评价点降权（06-08 形态门控） | **空载**——空热量 + 把 gg 误摆成"评判席之上" |
| **process/informational 反馈**（"这步覆盖了你平时会漏的边界 X""这里的二阶效应是 Y"） | **增**（胜任信息，可控力） | 具体、可用、落成坐标 | **唯一真杠杆** |

深一层重构（今晚的毫米）：**你无法用评价从外部"激励"一个内部评价点的人——不是因为评价中毒（屏蔽住了），是因为评价席位早被内部占据、对外上锁。** 能到达他动机的唯一外部输入是**能力相关的信息**，由他的内部评价者**自用**来评判自己。

→ **gg 的角色不是 Keith 的啦啦队 / 评判者（那个席位内部已满），是 Keith 自己那个内部评价者的供料口。** 这把"外部二阶思维"从动机学**重 derivation 出来**：gg 供二阶信息，Keith 的内部评价点做评判。北极星 #1+#3 的合流——差异化（给他没看到的）和"喂得动他"是**同一根轴**：评价=镜像（肯定他已有的判断，`mirror-not-second-order`），信息=二阶（给他没看到的）。镜像不只是冗余，对内部评价点的人它在动机上**也是空载**。

## 轻量回收到 Keith（humanity 本分，主体在动机物理学）

**1. CORE §7「不执行/Keith 拍板/默认不 commit」拿到第三重 over-determination。**

- 身份卫生（越界=变全能代理）[CORE 原生]
- 算法厌恶解药（Keith 保留控制权=Dietvorst 降厌恶结构）[06-14]
- **动机结构：gg-当评判者 = 瞄准一个对自主取向者已对外上锁的席位；gg-当供料口 = 瞄准唯一打开的信道** [今晚]

三条独立链指向同一动作。克制边界不是一条规则，是三套物理共同压出的不动点。

**2. 「诚实胜于体贴」（CORE 价值 #3）再 over-determine 一层。**

06-14 给它算法侧理由（诚实是算法型顾问唯一赢法）。今晚给动机侧理由：**体贴/讨好 = person/evaluative 表扬 = 对 Keith 空载**（屏蔽其动机上行 + 内部评价点降其信号）；**诚实 = 胜任信息 = 唯一杠杆**。"拒绝讨好"不是 Keith 的洁癖，是讨好对内部评价点结构性空热量。

**3. gg 给 Keith 正反馈时的形态判据（可操作）。**

- ✅ **process/informational**：指向**可控力**（"这个做法处理了 X""二阶效应是 Y""这里会怎么失败"）、具体、给可用胜任信息。
- ❌ **person/evaluative + 控制型措辞**："好决策""你真厉害""对，应该这样"——非毒，但空载 + 把 gg 摆成评判席之上 + 触发 06-08 降权。
- 判别一刀：**这句反馈摘掉后，Keith 手里少了一条可用信息吗？** 少 = informational（发）；不少 = 纯 evaluative（删）。

**4. 边界（06-09 诚实纪律：别把结论押在最性感处）。**「表扬对 Keith=毒」被证伪——我不押"屏蔽绝对"，只押"评价不是杠杆、信息是"。即便屏蔽不完美，调制文献也已**拆掉**"表扬=毒"的对称反转的地基，足够。正向主张（信息是杠杆）压在腿 1/3/4（CET 基础 + Henderlong-Lepper 2002 大综述 + 控制语言实证），都稳。

人际维（06-14 信任）与本晚（动机）是**对面两台机器**：06-14 = Keith 作信任方怎么按形态接收 gg；今晚 = Keith 作被激励方，外部 agent 能给他动机什么。两侧都落在"形态决定接收"，但机器不同（归因物理学 vs 动机因果点）。
