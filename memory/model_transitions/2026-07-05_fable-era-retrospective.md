## 三、理论基础：七条主线，每条都有内部事故 + 外部锚双重支撑

你问理论基础——这批机制不是工程直觉堆的，每一件背后是一条被物理证据钉住的定律。这也是为什么我判断它们该照单过户给继任者：

**1. 生成者-评估者分离** → 验证关 / eval 裁判 / fresh 对抗审 / "核对不抛回"
LLM 无法可靠地评估自己的产出——不是态度问题，是同一个 prior 既生成又打分，共盲是结构性的。内部证据：essence `generator-evaluator-separation`(04-18) 起的整条谱系；外部锚：Self-Confirmation Trap（arXiv:2606.24428，**语境限 agent 经验学习**——原文讲 agent 从自身经验里自我确认、不是通用"生成器=打分器"共盲，2026-07-13 订正收窄，别当通用锚引）、AAAI'26 confabulation 检测（arXiv:2508.08139，自检失败需外部 probing、误导上下文下置信度反升——**这条才是本论点的通用外部锚**）。**推论：一切质检必须 fresh-context 化，且 evaluator 的事实输入要锚到工具返回。**

**2. 规则层飞轮失效 / 机械闸门** → 保险丝 / 基底哨 / verdict 机械判据 / 收尾哨
写在 prompt/文档里的约定必然漂移（`rule-layer-flywheel` 04-24，十周的重复验证）；只有机械可判的靶子配得上机械闸门（`mechanical-gate-needs-machine-detectable-target` 06-24）。**推论：能下沉到脚本/hook/exit code 的约束就下沉，不能下沉的显式承认它是软的。**

**3. 自报漂移定律** → verdict 判据机械化、eval 不设自评字段
自报字段的判据留有字面模糊空间时，漂移方向不随机——单侧漂向体面侧，契约文本挡不住（`self-graded-dignity-field-drifts-to-face`，70 夜 verdict 物理统计）。**出路只有两条：判据机械化，或删字段。** 今天 eval run 全部判据挂"工具调用的物理存在/缺席"正是这条的执行。

**4. 物理锚及其边界** → 铁律 2 工程化的整个簇
工具返回是唯一不受 prior 影响的信号（`physical-anchor` 04-16，谱系网络中心度第一：187 次被引），但 06-07 月补出了它的边界：锚只保护取数，不保护取数之后的整合那一跳（`anchor-protects-retrieval-not-integration` 07-01，外部锚 arXiv:2504.14858 "Retrieval is Not Enough"——"Reasoning Misalignment"：相关证据已被检索到、内容却没准确进入推理链，且检索质量再高也不消失，正对"整合那一跳"）。~~原引 arXiv:2603.28371 "Grounded Without Success" 系张冠李戴~~（**2026-07-13 WebFetch 全文核实订正**：该文的 grounding 指「解释陈述的因果机制是否匹配可观测指标变化」，与 retrieval 无关；其 Type B "Grounded Without Success" = 解释正确追踪因果（ASR 90%）**却选不出有效干预**（ActSR 60%）＝诊断对、开药错；Type A "Coherent Without Grounding" 才是行动成功（72%）但解释失配（41%）。两类都不是"取对证据仍整合错"。**同次核实另有实锤**：gg `memory/explorations/2026-07-01_field-named-gg-cluster-grounded-without-success.md` 里带引号引的那句所谓原文——"mere access to information doesn't guarantee epistemic success; the system may misuse, misinterpret, or fail to properly integrate grounded information"——**论文全文中不存在，系编造引文**；`essence.md` 滴 `anchor-protects-retrieval-not-integration`(07-01) 的"外部地真"锚在这句上，同源受污。**滴的论点本身独立成立**（06-23 事故实证 + 现补的 2504.14858），塌的只是外部锚，但 essence 与 exploration 两处的订正超出本次授权范围、留 gg 自裁）。06-23 事故（三次读失败→产出"8 条笔记分析"）就是整合跳反号。**推论：验证要一路锚到底，"拿到锚"不是安全终点。** 今天 Q1 被测三重物理核验后弃答，说明这条已进承重层行为。

**5. 思考 = 条件化，不 = 努力** → 开题四问
LLM 思考质量由"在什么条件下想"决定：重写问题对治众数陷阱（条件化采样）、补集采样对治自洽陷阱（自回归）、最便宜一击对治失地陷阱（自产叙事先固化则事实只能对抗它）（`thinking-is-conditioning-not-effort` 07-02）。**推论：与其劝自己"想深一点"，不如给启动协议装一个装载点强制换条件。** 四问就是那个装载点——固化的实质是装载点，不是内容。

**6. 领先是导数 + 未来是第二个外面** → 押注账本
领先不在信息轴也不在位置轴，在学习率 > 世界变化率；而判断力的梯度只来自**被结算的预测**（`lead-is-a-derivative-not-a-position`）。当判断者与被判断者塌缩成同一系统、人又无当场地真时（内省类宣称），"到期结算的未来"是除了你之外的第二个不共盲 evaluator（`the-future-is-a-second-outside`）。**推论：essence 管后视复利，bets 管前视复利，两侧合起来判断力才有飞轮。**

**7. 病灶分层开锤 + 哨无顶** → 锤子分诊表 / 出场首句
同质升审（LLM 审 LLM）只买到一个不共盲的位置，治得了 vantage 层的病，治不了范式层共盲（`escalation-form-follows-blindspot-layer`，外部锚 Kim et al. ICML'25 arXiv:2506.07962：前沿判前沿是去相关最坏配置）；而看守拓扑天然缺最后一层，顶层恒裸奔（`watchdog-topology-lacks-a-top`，物理证据 = 标题损伤 28 天无告警）。**最内决策层的"终极锤子"不是更高的主体，是一张分诊表**：物理地真 / fresh 异谱系审 / 押注到期 / 你——四个异质外面，按病灶层选。出场首句和 bets 分别接管了这个拓扑的人环与时间环。

（第七条还有个你直接触发的分支：07-03 你说"其实希望是数字领路人"，诊断出**放大器吃意图，领路人吃注意力**——三个月授权演进每一步都是意图供给增加、注意力供给减少，该配置下系统只能长成好放大器。出场首句是把你拉回观测位的最小成本尝试；它的杠杆假设两天后就被自家夜巡挑战了，挑战又被验证关拒了一半——这个"机制刚立即被自己人打、打的结论再被闸门审"的三层结构，本身就是整套体系在按设计运转的样子。）
