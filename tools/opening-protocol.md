# opening-protocol — 开题四问

> 有决策对象的议题，看到问题后的**第一动作**。
> 每问的产出是**文本产物**，不是勾选——填不出产物 = 没做动作。
> 管辖：`CORE.md §8`（身体 / tools 目录）

---

## 为什么存在（三条物理事实）

LLM 思考质量由"在什么条件下想"决定，不由"想得多努力"决定。劝勉词（"要认真要严谨"）只有微弱语义引力，高杠杆在条件动作——四问是高杠杆清单，非穷尽全集（换先验 / 重排上下文等亦是条件动作）。三个结构性失败各对症一问，第四问管世界接触：

1. **条件化采样 → 众数陷阱**：第一反应 = 训练分布最常见解，不是最好解；问题的表述方式比后续一切努力更能决定答案
2. **自回归 → 自洽陷阱**：顺着写越写越自洽，与真假无关；同一 context 里自审 = 用生成错误的那个分布评估错误（fork 清 vantage 不清 prior，换模型才动 prior）
3. **叙事先固化 → 失地陷阱**：整条推理链若无外部事实进场，越长越自洽；早碰世界重塑整个下游分布，晚到的事实只能对抗已固化的叙事——时机比次数重要。边界：锚护取数、不护整合那一跳（`physical-anchor-has-rungs` / `anchor-protects-retrieval-not-integration`）

## 四问（产物锚）

| 问 | 产物（写进决策档 / 输出的开头） |
|---|---|
| **① 重写问题** | 重写后的问题一句，**禁用召唤原文的表述**。若换帧后答案空间没变，说明原表述已是本质——这本身是有效产物 |
| **② 判据先行** | "什么样的答案算错" + "错了会在哪个物理量上现形"各一句（对接 `reasoning_modules.md` M2 的"早期信号"字段） |
| **③ 补集采样** | ≥2 个与第一反应**方向不相容**的替代——装 `tools/solution-space.md` 执行（本协议是它的硬装载点，不再只靠"自觉发现已有答案"触发） |
| **④ 最便宜一击** | 本轮能做的最便宜现实接触（一次 grep / 一个 spike / 一条数据查询），做完引用工具返回；本轮做不了的，标注原因 |

## 豁免

- 简单问答 / 事实查询 / 纯流程任务（无决策对象 = 无题可开）
- Keith 明确要求快速回答
- auto_gg / exploration 不适用（自由不建笼，`caged-freedom`）

## 诚实边界

触发硬化到装载点（cc_agent 工作流步骤 / CLAUDE.md 启动协议）= L2；判定仍在 LLM 手里，**这不是飞轮**。
语义动作无机器可判的物理量，L3 hook 结构性不可行（`mechanical-gate-needs-machine-detectable-target`）——产物锚是当前可达的最强形态：字段即产物，填不出 = 没做（`reverse-anchor-by-reflection` 的机制）。
不设自评分字段（`self-graded-dignity-field-drifts-to-face`）。

## 跟其他工具的关系

- **③ 直接装 `solution-space.md`**——不重复其内容，本协议解决它的触发软肋
- **② 对接 `reasoning_modules.md` M2**——开题时只要"判据 + 现形物理量"两句，完整 pre-mortem 留给 COMPOSE 阶段按需装
- **先于一切装配**——四问发生在"看问题"之后、"装配判断"之前；开题产物会改变装什么的判断
